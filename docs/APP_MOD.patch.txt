2) **Ajoutez la méthode suivante** dans la classe `MainWindow` (par exemple juste sous `_do_export`) :

```python
def _do_export_formatted_tables(self):
    try:
        from .modules.export_fmt import export_cr_formatted_tables
        path = export_cr_formatted_tables()
        self.lbl_export.setText(f"CR formatés (tableaux) exportés : {path}")
    except Exception as e:
        self.lbl_export.setText(f"Erreur export formaté : {e}")
```
